# 欢迎来到AstrConter内核项目
这是一个基于Uinxed-Kernel的开源内核，因原项目为GPL-3.0协议，所以该项目也会基于GPL-3.0 协议开源 

# 环境搭建
## ArchLinux
```bash
sudo pacman -Sy make gcc clang
```
## Debian & Ubuntu & Kali
```bash
sudo apt update && sudo apt install make apt clang
```
如果您是Windows用户，可以使用Cygwin或WSL搭建环境并编译
